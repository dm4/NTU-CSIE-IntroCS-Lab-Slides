Zen Coding

  • Zen Coding

  • Zen Coding for Vim has its own ZC engine implementation.

  • A powerful abbreviation engine for HTML/CSS

Zen Coding

div#page>div.logo+ul>li#$$*3>a

can be expanded into:

<div id="page">
    <div class="logo"></div>
    <ul>
        <li id="01">
            <a href=""></a>
        </li>
        <li id="02">
            <a href=""></a>
        </li>
        <li id="03">
            <a href=""></a>
        </li>
    </ul>
</div>
