Shell

  • Interface between user and linux kernel

  • Many kinds of shells

    • bash
    • tcsh
    • csh
    • ksh
    • zsh
    • fish
    • ...

Shell

  • What shell you are using

    $ echo $SHELL

  • Change shell

    $ chsh

Wildcard

  • Wildcards will be expand before executing the command

  • List all txt files

    $ ls *.txt

    will expand to

    $ ls a.txt b.txt c.txt

Command Substituion

  • Shell replaces the backquotes (``) with the output of the command inside the backquotes

  • Create file with date

    $ touch `date +%F`.txt
$ ls
2011-10-10.txt

alias

  • Replace a long command to a short name

  • Some examples (using bash)

    • alias ll="ls -l"
    • alias la="ls -a"
    • alias rm="rm -i"
    • alias cp="cp -i"
    • alias ptt="ssh bbs@ptt.cc"
    • alias csie="ssh r00922006@linux15.csie.org"

Standard I/O

  • Standard input (stdin): keyboard

  • Standard output (stdout): screen

  • Standard error output (stderr): screen

  • Most UNIX tools use stdin and stdout.

  • Standard I/O allows you to redirect I/O away from your terminal to a file.

I/O Redirection & Pipeline

  • Redirect stdout to file

    $ command > file

  • Redirect stderr to file

    $ command 2> file

I/O Redirection & Pipeline

  • Append stdout to file

    $ command >> file

  • Pipe stdout to next Command

    $ command | command

date

  • Usage

    $ date [+output_format]

  • Some examples

    $ date
一 10月 10 14:52:53 CST 2011

date

  • Some examples

    $ date +%Y-%m-%d
2011-10-10


$ date "+%Hhr, %Mmin, %Ssec"
14hr, 56min, 07sec

  • More Information

    $ man date

tar

  • Usage

    $ tar [option] [file]

  • Option

    • c, create tar file
    • x, extract tar file
    • f, filename
    • v, verbose
    • z, compress with gzip
    • j, compress with bzip2

  • On Windows

    You can use 7-Zip

tar

$ ls
dir/

$ ls dir
A  demo  file

$ tar cf dir.tar dir
$ ls
dir/  dir.tar

$ mkdir extract
$ tar xf dir.tar -C extract/
$ ls extract/
dir/

head

  • Usage

    head [option] [file]

  • Print first n lines/bytes of file (default 10 lines)

head

  • First 5 lines

    $ head -n 5 filename

  • First 5 bytes

    $ head -c 5 filename

tail

  • Usage

    tail [option] [file]

  • Just like head

  • -F to output appended data as the file grows

du - disk usage

  • Usage

    du [option] [file]

  • Option

    • -h, human readable
    • --max-depth=n

find

  • Usage

    find [path] [option] [action]

  • Option

    • -name
    • -user
    • -group
    • -mtime, -atime, -ctime

find

  • Action

    • -exec ... {} \;

  • Powerful command to find files!

find

  • Find all txt file

    $ find . -name "*.txt"

  • Find all txt file and print content

    $ find . -name "*.txt" -exec cat {} \;

grep

  • Usage

    $ grep [option] pattern [file]

  • Option

    • -i, ignore case
    • -n, show line number
    • --color=auto

top

  • Usage

    $ top

  • Display tasks information
