Introduction to Vim

Introduction to Vim

  • Vim

    Vi IMproved, a programmers text editor

  • What Is Vim?

    Vim is a highly configurable text editor built to enable efficient text editing. It is an improved version of the vi editor distributed with most UNIX systems.
    Vim is often called a "programmer's editor," and so useful for programming that many consider it an entire IDE. It's not just for programmers, though. Vim is perfect for all kinds of text editing, from composing email to editing configuration files.
    ~ Vim online

Modes in Vim

  • Normal Mode

    • move cursor
    • copy & paste
    • delete

  • Insert Mode

    • edit text
    • press i to get into Insert mode
    • press <ESC> or <Ctrl-C> to go back to Normal mode

Modes in Vim

  • Visual Mode

    • select text
    • press v to get into visual mode
    • press Ctrl-v to get into block-visual mode
    • press <ESC> or <Ctrl-C> to go back to Normal mode

  • Command Mode

    • enter vim commands
    • :[command]

Open File

  • In command line

    $ vim [filename]

  • In vim

    :e [filename]

Save File

  • Save

    :w

  • Save as [filename]

    :w [filename]

Exit Vim

  • Exit

    :q

  • Exit without save

    :q!
ZQ

  • Save & Exit

    :wq
:x
ZZ

Copy & Paste & Delete

  • In Normal Mode
  • Copy
    • yy to copy current line
    • 3yy to copy the following 3 lines
  • Delete
    • dd to delte current line
    • 3dd to delte the following 3 lines
  • Paste
    • p to paste below the current line
    • P to paste above the current line

Undo & Redo

  • Undo

    • u to undo

  • Redo

    • <Ctrl-r> to redo

Search

  • In Normal Mode
  • /[pattern]
  • n to next result
  • N to previous result

Replace

  • in {range} replace the first occurrence of the word foo with bar with {flags}

    :{range}s/foo/bar/{flags}
  • range
    • % for whole file
    • 123 for line 123
  • flags
    • g for replace all occurrences in {range}
    • c for confirm before each substitution
    • e for ignore case
    • :help :s_flags
