What's regular expression?

  • Regular expression(regexp)
  • Special syntax to do
    • string matching
    • data extraction
    • string replacement

Metacharacters

  • Basic regular expression(BRE)

    ^   $   .   *   \(   \)   [   \{   \}   \

  • Extended regular expression(ERE)

    |   ^   $   .   *   +   ?   (   )   [   {   }   \

Metacharacters

  • []

    Used for specifying a character class, which is a set of characters that you wish to match.

    [aeiou]
[a-zA-Z0-9_]

  • [^]

    Matches the characters not listed within the class by complementing the set.

    [^aeiou]

Metacharacters

  • \

    Escape all the metacharacters so you can still match them in patterns.

    \\                  # match \
\[                  # match [

Metacharacters

  • .

    Matches any single character

    a.c                 # can match 'abc', 'axc', ...

  • ^

    Matches the start of the line

  • $

    Matches the end of the line

Metacharacters

  • |

    Alternation, or the "or" operator

    a.c|a.b         # can match 'abc', 'acb', 'axb', ...

  • ()

    • Defines a marked subexpression

    • or \(\) in basic regular expression

      (cat|dog)*

Repeating

  • {m,n}

    • Matches the preceding element at least m and not more than n times.
    • or \{\} in basic regular expression

  • *

    • {0,∞}

  • +

    • {1,∞}

  • ?

    • {0,1}

Regular expression in Vim

  • Using

      • [a-zA-Z0-9_]
