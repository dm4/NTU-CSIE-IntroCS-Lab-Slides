Data Structure

  • List

    • An array with variable length

  • Tuple

    • An list but immutable (constant)

  • Dictionary

    • An array but use string as index

List

list1 = [5,3,6,1,4,2]
list2 = list1

Since we assign “list1” to “list2”, “list2” will reference the same object of “list1”.

list1.sort()
print list2
# [1, 2, 3, 4, 5, 6]

Tuple

Tuple is immutable, so you can’t sort (modify) it.

tuple1 = (1,3,2)
print tuple1
# (1, 3, 2)
tuple1.sort()
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'tuple' object has no attribute 'sort'

Once if you want to sort your tuple, you have to copy it, modify it, and restore it.

Dictionary

A dictionary is made by “{}” syntax. If you typed “{}”, you’ll get an empty dictionary.

dict1 = { "name":"Mike", "gender":"Male"}
print dict1["name"]
# Mike

You can add key-value pair in to a dictionary like line 2. Just define key and assign value directly.

dict1["job"] = "professor"
print dict1
# {'gender': 'Male', 'job': 'professor', 'name': 'Mike'}

Dictionary (cont.)

Get or update value is quiet easy. You can use “pop” to remove pair in dictionary.

dict1["name"] = "Mike Chen"
dict1.pop("job")
# 'professor'
print dict1
# {'gender': 'Male', 'name': 'Mike Chen'}

Function

def cumsum(n, maximum=10, minimum=1):
    sum = 0
    for i in range(minimum, min(n, maximum)):
        sum += i
    return i, sum   # Return values can be more than 1!

You don't need to pass arguments which has default value.

cumsum(20)
# (9, 45)
cumsum(maximum=5, n=20)
# (4, 10)

Function (cont.)

Return values can assign to variable separately.

a, b = cumsum(20)
print a
# 9
print b
# 45
